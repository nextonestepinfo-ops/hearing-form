# Premium Motion Site Template

高単価プラン用のリッチ演出サイトデモです。100万円級の提案時に「ここまで動かせる」を見せるため、通常テンプレートより演出量を増やしています。

## 使いどころ

- キャバクラ、ラウンジ、バーなどのナイト店舗
- カフェ、飲食店など雰囲気を見せたい店舗
- Vtuber、配信者、タレント、アーティストの公式サイト

## 差し替えるもの

- 店名、ブランド名
- 写真、動画、ロゴ、色味
- 予約、LINE、SNS、電話、問い合わせ先
- メニュー、料金、出勤、配信予定、出演実績

## 営業メモ

- 通常テンプレートより「世界観」「動き」「写真の見せ方」を重視する高単価向け。
- 同じ土台で Lounge / Cafe / VTuber / Talent の文言に切り替えられる。
- 実案件ではヒアリング内容から、業種に合わせて細かく構成、導線、更新項目を調整する。
- ローディング、常時動く背景、マウス追従、スクロール連動の横移動、業種切替アニメーションを入れている。
- 写真や動画素材が十分にある案件では、動画背景、人物別ページ、スケジュール更新、LINE/予約連携まで広げる。
