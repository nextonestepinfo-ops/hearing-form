# Vtuber用サイト デモ

活動中の女性VTuber向けに、配信予定、プロフィール、推し活リンク、グッズ、企業案件相談を1ページにまとめる公式サイトデモです。

## 公開URL

https://nextonestepinfo-ops.github.io/hearing-form/vtuber-kawaii-site-template/

## 営業時の見せ方

- すでに活動している人ほど、YouTube、X、グッズ販売、案件相談が分散しやすいことを起点に話す。
- サイト側で情報を整理すると、ファンにも企業にも見せやすくなると伝える。
- 立ち絵、三面図、表情差分、配信サムネ、グッズ画像までまとめて作れる例として見せる。
- 実案件では名前、配信内容、リンク、素材、問い合わせ先を差し替える。

## ファイル

- `index.html`: サイト本体
- `styles.css`: デザインとアニメーション
- `script.js`: タブ切替、スクロール演出、カーソル演出
- `assets/`: キャラクター・素材イメージ
