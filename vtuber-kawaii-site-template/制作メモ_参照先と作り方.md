# 制作メモ_参照先と作り方

## 目的

活動中VTuber向けに、公式サイト、配信導線、グッズ導線、企業案件相談の入口をまとめて見せる営業デモとして制作。

## 参照した考え方

- VTuber公式サイトは、キャラクターの世界観と活動情報が同時に伝わることが重要。
- ファン向け導線と企業向け導線を分けすぎず、同じページ内で自然に移動できる構成にする。
- 派手な演出は、営業相手が内容を読める範囲に抑える。

## 制作内容

- ローディング、スクロール進捗、フェードアップ、チルト、スパークルを実装。
- 通常配信、ゲーム、歌枠の切替で、活動内容に合わせた見せ方を提案できるようにした。
- 立ち絵、三面図、表情差分、配信バナー、グッズモックアップをSVG素材として作成。
- GitHub Pages公開用に、軽量で壊れにくいテキストベースの素材に整理。

## 使い方

1. デモ一覧から `Vtuber用サイト` を開く。
2. ファーストビューで、活動情報をまとめる価値を説明する。
3. `素材` セクションで、サイト以外の制作物も一緒に提案できることを見せる。
4. `Contact` セクションで、案件相談や制作相談フォームへつなげる。

## 営業トーク例

「すでに活動している方ほど、YouTube、X、グッズ販売、案件相談が別々になりがちです。公式サイト側に活動情報を整理し、必要に応じて立ち絵、三面図、表情差分、告知ビジュアル、グッズ画像までまとめて作ることで、ファンにも企業にも見せやすい状態にできます。」
