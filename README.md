## メモ

- フォーム送信は `ヒアリング入力` タブへ追記します。
- 取引先ポータルは `取引先ポータル` タブを読み、`取引先やり取り` タブへ連絡を追記します。
- デモサイトは営業時に見せるサンプルです。写真、メニュー、営業時間、予約導線、問い合わせ先を案件ごとに差し替えて使います。
- 高単価リッチ演出サイトは、ラウンジ、カフェ、Vtuber、タレント系など、世界観と動きで見せたい案件用です。
- Vtuber用サイトは、活動中VTuber向けに配信、グッズ、推し活リンク、案件相談をまとめる公式サイト提案用です。
