# 白狼このは公式サイト V5 制作メモ

## 目的

白狼このはさんの現在の方向性を保ったまま、プレミアムVTuber公式サイトとして完成度を上げる。

## 実装方針

- 背景は完成イラストではなく、UIが乗る空間として扱う
- キャラクターが主役になるよう、背景の主張は控える
- `moon / live / silver` の3シーンでスクロールに合わせて空気感を変える
- 白、ミント、シルバー、透明感、柔らかい光を維持する
- ガラスUI、細いライン、霧、粒子、反射で高級感を出す
- スマホでのファーストビューとスクロール体験を最優先にする

## シーン設計

- `hero / profile`: moon。白月光、淡いミント、キャラを引き立てる余白
- `links / movie`: live。配信感、薄いグリッド、軽い動き
- `tags / contact`: silver。銀白、落ち着き、ブランド締め

## 注意

- 過剰なサイバーHUDにしない
- 背景素材を強く見せすぎない
- UIの可読性を優先する
- 余白を削らない
- 動きは常に静かで上品にする
