# 白狼このは公式サイト V5 Premium Scenes

既存の白狼このは公式サイトをベースに、プレミアムVTuber公式サイト体験として仕上げたV5版です。

## V5の狙い

- 3枚の空気感背景を使ったセクション連動のシーン切替
- `moon / live / silver` の固定背景ステージ
- V5専用生成背景 `v5-scene-moon.jpg` / `v5-scene-live.jpg` / `v5-scene-silver.jpg`
- 粒子、霧、ガラス反射、細いライン演出
- スマホ優先の余白、可読性、ガラスUI調整
- 各セクション到達時の上品なフェード・スケール表示
- キャラクターを主役にした静かな高級感

## 主要ファイル

- `index.html`: V5公式サイト本体
- `styles.css`: V5プレミアム背景、ガラスUI、レスポンシブ調整
- `script.js`: パスワードゲート、ロード演出、スクロールシーン切替、YouTubeサムネ反映
- `site-config.json`: 名前、素材、SNS、タグ、YouTubeサムネ設定
- `youtube-archives.json`: 直近のYouTubeアーカイブ一覧
