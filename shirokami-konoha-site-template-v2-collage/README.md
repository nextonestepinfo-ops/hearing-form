# 白狼このは公式サイト V2 Collage

既存版 `shirokami-konoha-site-template` を残したまま作成した、別バージョンの公式サイトです。

## 方針

- デモサイト表記は使わず、白狼このは公式サイトとして構成
- トップはロゴと大きな立ち絵を中心に配置
- `IMG_2622.png` をプロフィールのメインビジュアルとして使用
- ミント、シルバー、ステッカー、ピクセル、ウィンドウUI、キラキラ、リボン、肉球チャームの方向へ調整
- キャラの新規生成は行わず、既存素材と提供画像を使用
- GIFは使わずCSSアニメーションで演出
- スマホ表示を優先して調整

## 主要ファイル

- `index.html`: トップ、プロフィール、公式リンク、タグ、お仕事相談
- `styles.css`: V2のコラージュUI、レスポンシブ、アニメーション
- `script.js`: パスワードゲート、ロード画面、メニュー、表示アニメーション
- `assets/konoha-collage-profile.jpg`: プロフィール用の軽量メイン画像
- `assets/konoha-collage-profile-mobile.jpg`: スマホ用の軽量メイン画像
- `assets/konoha-collage-original.png`: 提供画像の原本コピー
- `preview-v2-desktop.png`: デスクトップ確認用プレビュー
- `preview-v2-mobile.png`: スマホ確認用プレビュー
